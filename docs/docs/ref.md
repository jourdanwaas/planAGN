References and Acknowledgments
===============

The [planAGN](https://github.com/jourdanwaas/planAGN) model is an evolution of the work done by [Balbi & Tombesi (2017)](https://www.nature.com/articles/s41598-017-16110-0) and [Ambrifi et al. (2022)](https://academic.oup.com/mnras/article/512/1/505/6539336). 

The specificities regarding the mathematical development of the model are detailed between these two publications and the aforementioned associated publication by [Waas et al. (2026)](https://iopscience.iop.org/article/10.3847/1538-4357/ae5e6f).

---

Public release of this code is available following the most recent publication, therefore, if you use [planAGN](https://github.com/jourdanwaas/planAGN), we would appreciate an acknowledgement toward the pioneering work that allowed this model to come to fruition. We suggest either the following paragraph, or some variant.

(2022)](https://academic.oup.com/mnras/article/512/1/505/6539336), and [Waas et al. (2026)](https://iopscience.iop.org/article/10.3847/1538-4357/ae5e6f).
